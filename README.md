# img_st_x

Dynamic Open Graph & Twitter meta image rendering by path (Next.js, Vercel ready)

## Usage
- Place your images in `/public` (e.g. `img1.png`, `img2.png`, `img3.png`)
- Access `/img1`, `/img2`, `/img3` to get different meta images
- Deploy to Vercel for best compatibility with X/Twitter/Facebook

## Example paths
- `/img1` → shows meta for `img1.png`
- `/img2` → shows meta for `img2.png`
- `/img3` → shows meta for `img3.png`

## Custom domain
- Point your Namecheap domain to Vercel (CNAME)
- See Vercel docs for details
